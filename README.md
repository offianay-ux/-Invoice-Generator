# 🧾 Invoice Generator

A clean, minimal invoice generator web app built with Python and Flask. Fill in your details, add line items, and download a professional PDF invoice instantly — no subscriptions, no third-party tools.

---

## ✨ Features

- Input business & client details
- Add multiple line items dynamically
- Live subtotal, tax & total calculation
- Multiple currency support (INR, USD, EUR, GBP)
- Generates & downloads a PDF invoice instantly
- Liquid glass UI with gradient background

---

## 🛠️ Tech Stack

- **Python & Flask** — backend & routing
- **ReportLab** — server-side PDF generation
- **HTML / CSS / JS** — frontend
